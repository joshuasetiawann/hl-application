# Premium UI — Screenshots

Generated by `npm run screenshots` (script: `scripts/screenshot.cjs`).

> These PNGs are **not** captured automatically in the cloud/CI environment because
> the browser download host is blocked there. Generate them locally in seconds.

## How to generate

```bash
# 1. Start the app
./run-linux.sh            # Windows: run-windows.bat

# 2. Make a browser available to Puppeteer, then run the capture:
#    Option A — use a Chrome/Chromium already on your machine:
PUPPETEER_EXECUTABLE_PATH="$(command -v google-chrome || command -v chromium)" npm run screenshots
#    Option B — let Puppeteer download its own browser once:
npx puppeteer browsers install chrome && npm run screenshots
```

Output PNGs land in this folder (`screenshots/premium-ui/`).
